# Project Prompts

## 中文原始提示词

以下内容整理自本次项目推进中最关键的用户指令：

1. 根据这个，做一个windows下可以运行的软件，输入word输出md。 软件最好是绿色不需要安装直接运行那种
2. 重新生成zip包加上版本号
3. 生成一段介绍英文，简单点精炼点。但是需要图文并貌。帮我上传到git，发布上去。名字叫windows_word_to_md吧
4. 再发布个git，名字叫做AI_to_windows_word_to_md. 介绍内容就是刚才的通过codex利用windows发布的word2md做好的，代码就是我的提示词。翻译成中英双语

## English Translation

Below is a cleaned-up translation of the key prompts that shaped this project:

1. Based on this, make a Windows app that can run locally, takes Word as input, and outputs Markdown. It should preferably be portable and run without installation.
2. Rebuild the zip package and add a version number.
3. Write a short and polished English introduction. It should include both text and visuals. Help me upload it to Git and publish it. Name it `windows_word_to_md`.
4. Publish another Git repository named `AI_to_windows_word_to_md`. The introduction should say that the Word-to-Markdown project was built through Codex collaboration for Windows, and the code here is my prompt itself. Translate it into both Chinese and English.
